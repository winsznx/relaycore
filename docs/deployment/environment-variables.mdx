## Required Variables

| Variable | Description |
|----------|-------------|
| `VITE_SUPABASE_URL` | Supabase project URL |
| `VITE_SUPABASE_ANON_KEY` | Supabase anonymous key |
| `WALLET_PRIVATE_KEY` | Wallet for signing |
| `ESCROW_CONTRACT_ADDRESS` | EscrowSession address |
